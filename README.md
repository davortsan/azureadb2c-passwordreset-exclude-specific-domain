# azureadb2c-passwordreset-exclude-specific-domain
<b>Disable password reset policy to specific domain</b>
<p>Whit these B2C polcies you could to reset your password for all users that are registered in your Azure AD B2C tenant but it is not going to be possible to those users that were registered using an specific domain. In this examplo, users with <b>domainnametoexclude.com</b> in their mails will not be able to reset the password.</p>
